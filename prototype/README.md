# 栄養計算プロトタイプ

`index.html` をブラウザで開くと、経腸栄養と静脈栄養の簡易計算プロトタイプを使えます。
`products_master_sample.csv` は Excel 移植用のサンプルマスタです。

## できること

- 製品名をプルダウン選択
- 入力枠は 5 行固定
- `mL/日` を直接入力
- または `mL/h × 時間/日` から 1 日量を自動計算
- 総カロリー、タンパク量、脂質、糖質、窒素量、NPC/N を集計
- スマホ向けカード表示
- PWA対応
- ホーム画面追加対応

## データについて

- 初期搭載の製品マスタは `★輸液栄養、経腸栄養まとめ.pdf` を見ながら手入力したサンプルです
- 実運用前に、採用品目に合わせて全製品を追加・再確認してください
- アミノ酸製剤は PDF 上の窒素量からタンパク相当量を算出しているものがあります

## スマホで使う

- Android: 公開URLを Chrome で開くと「ホーム画面に追加」または「インストール」が使えます
- iPhone: Safari の共有メニューから「ホーム画面に追加」を選びます
- PWA のオフラインキャッシュは `https://` または `http://localhost` で有効です
- `file://` で直接開いた場合、画面は表示できますが PWA 機能は一部動きません

## 公開URLについて

- このフォルダを GitHub Pages、Netlify、Cloudflare Pages などに置くとスマホからURLで使えます
- 公開URLはデプロイ先で決まります
- 例:
  - GitHub Pages: `https://<ユーザー名>.github.io/<リポジトリ名>/prototype/`
  - Netlify: `https://<任意のサイト名>.netlify.app/`
